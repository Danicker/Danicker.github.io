Changelog

The World's Hardest Game Website is an ongoing project that will receive updates over time. If you have any feedback or suggestions then please contact Danicker on discord.

Version 3.2 – 27th September 2024

  • Added GLITCH LIST to the WIKI page. This is a very comprehensive list of glitches, exploits and oversights in the Flash versions of The World's Hardest Game. Thanks to Fraims for compiling this list. This is currently the only page on the wiki but we hope to expand it in the future.
  • Some minor improvements to CUSTOM LEADERBOARDS:
    • Leaderboard categories will now always be up-to-date with the current categories available on speedrun.com. In particular, TWHG1 HTML5 individual levels, TWHG2 Glitched and TWHG3 Paint% are now accessible in the leaderboards system.
    • Player names will now be displayed in the most up-to-date user profile information.
    • Banned players will now appear in dark grey.
  • Some minor improvements to the ROUTE LIST:
    • Added URL parameters for the game and level, so you can share links like http://twhg.info/routes?game=2&level=15.
    • Added an info button to the top right corner that provides some additional information about the route list.

Version 3.1 – 23rd February 2024

  • Made some minor changes to GETTING STARTED to hopefully make it more accurate and more helpful.
  • Some minor improvements to CUSTOM LEADERBOARDS:
    • Fixed bug where times containing minutes would sometimes only show one digit for seconds.
    • Removed glitched categories for TWHG2 since those don't exist anymore.

Version 3.0 – 18th December 2023

  • Added ROUTE LIST page. This is a place where common routes for TWHG will be documented, and can be viewed and compared. There is currently only support for TWHG1 and TWHG2, and also currently only a limited number of routes. This is just the first version of the Route List and it will be improved over time. If you would like to help contribute to the Route List, contact Danicker on discord.
  • Some minor improvements to CUSTOM LEADERBOARDS:
    • Player names now appear in bold font, coloured according to their speedrun.com user profile.
    • You can now hover over some entries in leaderboard tables for extra information.
      • Hovering over the "Multiple" player tag in a Sum of Best table will display the players that contributed towards the community SoB (and how many categories they contributed to).
      • Hovering over the numbers in a Medal Count table will show which categories the player holds that medal in.
      • Hovering over a date will show how many years/days have passed since that date.
  • Removed old TWHG1 and TWHG2 custom leaderboards from Quick Links.
  • Added TWHG Texts and TWHG Colours Spreadsheets to Quick Links.
  • Changed http://twhg.info/discord invite to point to #help instead of #lobby.

  • Added TWHGEDIT3 to quick links.
