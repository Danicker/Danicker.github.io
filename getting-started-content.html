Getting Started

Introduction

The World's Hardest Game is a series of four flash games created by Stephen Critoph in the period 2007-2016. If you're a fan of TWHG then I recommend you join the discord server, where the community discusses anything and everything related to TWHG, including speedrunning and custom levels. We always like to have new members and if you have any questions we can help you out.

How To Play

So, you want to play The World's Hardest Game, but you don't know where to start. As of 2021, Adobe Flash is no longer supported in web browsers, however there are still three ways to play The World's Hardest Game:

1. Flash versions using Adobe Flash Projector

The recommended way of playing TWHG is to use Adobe Flash Projector. Adobe Flash Projector is the only official emulator of Adobe Flash. It emulates The World's Hardest Game (almost) exactly as it was experienced prior to 2021 in a web browser. This is the only acceptable method if you wish to submit your run to the main speedrun.com leaderboards (for the HTML5 leaderboard, use the HTML5 version as linked below).


To run TWHG in Adobe Flash Projector you will need to complete the following steps:

  • Download Adobe Flash Projector: Go to http://twhg.info/resources, open the Adobe Flash Projector Downloads folder and read the INFO file to find the right version to download for your operating system.
  • Download TWHG flash files: Download the flash file(s) you want to run from http://twhg.info/resources. Flash Files have the .swf file extension. In the resources folder you can find the official releases of TWHG, speedrun/practice mods and other mods.
  • Open your flash file(s) using Adobe Flash Projector: There are a few ways you can do this, if one doesn't work for you then try another way:
    • Right click on your .swf and then click "Open with" and select Adobe Flash Projector. If you then set Adobe Flash Projector as the default app to open .swf files you can open any other .swf files by just double clicking on them.
    • Open Adobe Flash Projector and then click and drag your .swf into it.
    • Open Adobe Flash Projector and then click File -> Open or Ctrl+O then select your .swf file.

If you are still having issues running flash files then join the discord server and ask for help.

2. Flash versions using Ruffle

Ruffle is an unofficial flash emulator. It does not emulate enemy collisions completely accurately, and hence is not allowed on the speedrun.com leaderboard. However, the advantage of Ruffle is that you can run it in your web browser without any downloads. There are many websites that use ruffle to run TWHG but here is one option you can use for TWHG1 and TWHG2:

  • TWHG1 Ruffle: https://www.armorgames.com/play/1043/the-worlds-hardest-game
  • TWHG2 Ruffle: https://www.addictinggames.com/puzzle/worlds-hardest-game-2
[Note: It is not recommended to run TWHG3 or TWHG4 in ruffle as ruffle does not support ActionScript 3.]

3. The HTML5 versions

Alternatively, you may choose to play the HTML5 versions (sometimes called 'Updates') of The World's Hardest Game. These can be played on coolmathsgames with no downloads however they are quite different from the original versions of the game. TWHG1u was made by Stevie and has its own leaderboard on speedrun.com. The other HTML5 games were not made by Stevie and have substantial flaws, TWHG2u and TWHG4u being impossible to finish. You can play the HTML5 versions here:

  • TWHG1 Update: https://www.coolmathgames.com/0-worlds-hardest-game/play
  • TWHG2 Update: https://www.coolmathgames.com/0-worlds-hardest-game-2/play
  • TWHG3 Update: https://www.coolmathgames.com/0-worlds-hardest-game-3/play
  • TWHG4 Update: https://www.coolmathgames.com/0-worlds-hardest-game-4/play

Speedrunning

Speedrunning times are tracked on the speedrun.com leaderboards for TWHG. If you want to speedrun TWHG, there are a few more things you will need:

  • Recording Software: You will need to have a video recording of your run if you wish to submit it to the speedrun.com leaderboards. OBS Studio is recommended for this.
  • Livesplit: This is not required for a speedrun, but can be helpful, especially if you are using a version of TWHG that does not have a timer. Livesplit allows you to time your runs and compare to your PB in realtime. Download livesplit here.
[Note: If you are submitting to the main leaderboards, you must use Adobe Flash Projector or Flashpoint. If you are submitting to the HTML5 leaderboard you must be playing the HTML5 version. Ruffle is not allowed on any speedrun.com leaderboard.]

Custom Levels

Several unofficial level editors have been made by fans of The World's Hardest Game. For quick links see the home page. For more information about how to play custom levels and how to make your own levels, join the discord server and check the pinned messages in the #editor channel.
